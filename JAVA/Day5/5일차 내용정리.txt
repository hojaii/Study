배열
primitive Data Type 변수들은 메모리에 저장된 하나의 값을 참조함
메모리의 저장된 데이터 집합을 하나의 객체(변수)로 참조가능

배열은 reference type(타입선언, new로 생성, 멤버들을 초기화함)

배열을 생성할때 데이터 저장갯수(size, 크기)를 함께 선언해야함

배열이 생성되면 자동으로 length멤버필드도 생성되고 여기에는 배열의 크기값이 생성됨

배열의 요소(item)에 참조하거나, 사용하려면 for반복문을 사용함
