While반복문
반복 수행 횟수를 모르고, 반복 수행 조건을 알고있을때 주로 사용함

while(조건표현식){
		반복수행문장;
		//반복수행 계속 조건,문장 선언가능
	           }

무한루프는 반드시 탈출조건과 함께 선언해야함

Do while 반복문
1번 실행후 계속 반복할지 여부를 결정함

Do {
	반복수행문장;
	.......
     } while 조건문 {
		}


Continue 
반복블럭내에서 특정조건이 true일때 반복수행하지않고 다음반복회차로 실행함

break
