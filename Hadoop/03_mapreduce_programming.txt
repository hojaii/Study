* 맵 과 리듀스
> 맵리듀스 프로그램의 특성
	: 맵과 리듀스 두단계로 구성되고 주어지는 데이터나 출력으로 내보내 지는 데이터가 모두
	키와 밸류로 구성됨

> 맵의 기본동작
	: 입력 데이터의 레코드를 하나씩 처리함
	입력 레코드는 하나의 키와 하나의 벨류로 구성됨
	주어진 입력 키와 벨류를 새로운 키와 벨류로 변환함

> 리듀스의 기본동작
	: 같은 키를 가진 레코드들을 묶고 하나의 리듀스 입력 레코드로 만들어짐
	리듀스는 또 다른 처리를 하고 새로운 키와 벨류를 출력함

> 맵리듀스 프레임워크의 역할
	: 입력 파일을 맵의 입력 레코드로 만들어주고 맵에서 출력된 레코드들에서 같은 값을 갖는 키의
	벨류들을 하나의 리스트로 묶어서 리듀스로 보내줌

> 맵리듀스 개발과정
	: 맵리듀스 단계별로 사용할 파라미터를 키와 값의 형태로 설계하고 
	Mapper 클래스를 상속받아서 WordCountMapper 클래스를 구현하고
	Reducer 클래스를 상속받아서 WordCountReducer 클래스를 구현하고
	맵리듀스를 잡을 객체를 생성하고 실행시키고 완성된 맵리듀스 클래스를 하둡으로 보내서 실행함



* 정렬
: 맵리듀스는 입력데이터의 키를 기준으로 정렬되므로 하나의 리듀스 태스크만 실행하면 정렬이 쉽게되고
이 종류로른 보조정렬, 부분정렬, 전체정렬이 있음
> 보조정렬
	: 키의 값들을 그루핑하고 그루핑한 레코드에 순서를 부여함
