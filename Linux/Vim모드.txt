Vim 모드
: 리눅스를 이용하는데 있어서 가장 기본적인 에디터

Vimtutor를 사용해서 각종 사용방법을 익힐수도있음

명령모드
: 키입력으로 vim에 명령을 내리는 모드

입력모드 
: 문서를 편집하는 모드 I키를눌러 활성화 , esc키를 눌러 비활성화

ex모드
: 에디터 기능을 사용하는 모드 : 키를 눌러 활성화함

$vi test
: test이름의 파일을 생성 or  열기

:w
: 파일을저장

:q
: 파일을 종료

:wq
: 파일을 저장한 후에 종료함

:wq!
: 파일을 강제로 저장한 후에 종료함

:w >>temp
: temp라는 파일에 현재 파일내용을 저장함

:e temp
: temp라는 파일을 불러옴
